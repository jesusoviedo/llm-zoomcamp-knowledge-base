### 5. MCP (Model Context Protocol): El Est√°ndar para un Ecosistema Conectado

MCP es la pieza que permite escalar las arquitecturas de agentes. Es un protocolo abierto que estandariza la comunicaci√≥n entre los modelos de IA y sus herramientas.
